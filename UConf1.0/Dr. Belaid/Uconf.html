  • Ce n'est pas un effet de mode!

Natives

Android SDK

Avantages

  • Accès optimisé aux devices du mobile (GPS, Caméra…)
  • Meilleur expérience utilisateur (UX)
  • Gestion native des évènements (touch, swipe, shink…)
  • Fluidité des applications Android

Inconvénients

  • Spécifique à la platforme.
  • Coût de développement élevé

Hybrides

Xamarin, Phonegap/Cordova, Titanium, Sencha, Ionic…

Avantages

  • Portabilité (one code base)
  • Coût réduit de développement
  • Développement rapide d'application

Inconvénients

  • Accès limité aux devices du mobile
  • Fluidité des applications
  • Gestion des évènements lente (300ms)
  • Une expérience utilisateur moyenne
